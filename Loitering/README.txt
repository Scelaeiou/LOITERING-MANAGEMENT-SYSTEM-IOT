# ESP32 RFID Timer Dashboard

This project integrates multiple ESP32 devices with RFID readers to manage countdown timers for different tables. The system features a PHP-based web dashboard that updates in real time and synchronizes with a MySQL database hosted on XAMPP.

---

## 💡 Features

- 🕒 **Real-time countdown timers** for 6 tables
- 📶 **ESP32 + RFID**: Tapping an RFID tag adds time to the table
- ⏳ **Timer reduces** every second and syncs across devices
- 🔔 **Buzzer alert** when timer hits zero
- 📋 **History tracking** of RFID scans and timer usage
- 🌐 **Web dashboard** built with HTML, CSS, JavaScript, and PHP
